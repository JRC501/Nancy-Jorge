# 💍 Nancy & Jorge - Invitación Web

Proyecto desarrollado como práctica de desarrollo web para crear una página de invitación digital para un evento.

## 📌 Descripción
Este repositorio contiene una página web diseñada como invitación para un evento especial. 
El objetivo fue aplicar conocimientos de HTML y diseño web para crear una experiencia visual sencilla y funcional.

## 🎯 Objetivos del proyecto
- Practicar estructura básica en HTML
- Diseñar una página web temática
- Aplicar estilos visuales para eventos
- Crear una invitación digital accesible

## 🛠️ Tecnologías utilizadas
- HTML
- CSS
- Diseño responsivo básico

## 📂 Contenido del repositorio
- Página principal de invitación
- Recursos visuales del evento
- Archivos de estilo

## 🌐 Demo en vivo
Puedes ver la invitación web funcionando aquí:  
👉 https://jrc501.github.io/Nancy-Jorge/

## 🚀 Vista previa
Este proyecto muestra una invitación digital que puede utilizarse como base para:
- Eventos sociales
- Bodas
- Celebraciones
- Invitaciones digitales personalizadas

## 📚 Aprendizajes
- Estructuración de una página web en HTML
- Organización de recursos para un proyecto web
- Publicación de sitios usando GitHub Pages
- Diseño básico para invitaciones digitales
